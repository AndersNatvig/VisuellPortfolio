Øyeblikksbilder

Event, miljø, stemning, mennesker

I denne serien handler alt om å fange stemningen slik den faktisk er. Latter, samarbeid, små blikk og ekte øyeblikk som forteller mer enn iscenesatte bilder noen gang kan. Disse bildene viser mennesker i samspill – naturlig, levende og ufiltrert. Jeg har ikke jobbet mye med slike bilder, men mer er på vei!

Dette området er lagd av Anders Natvig med HTML, CSS og Java script
